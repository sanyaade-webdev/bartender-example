# Open Beer Database Bartender Example [![Build Status](https://secure.travis-ci.org/tristandunn/bartender-example.png?branch=master)](http://travis-ci.org/tristandunn/bartender-example) [![Dependency Status](https://gemnasium.com/tristandunn/bartender-example.png)](https://gemnasium.com/tristandunn/bartender-example)

A basic example of using [Bartender](https://github.com/tristandunn/bartender) to interact with [Open Beer Database](https://github.com/tristandunn/openbeerdatabase).

## Setup

    git clone git://github.com/tristandunn/bartender-example.git
    cd bartender-example
    bundle
    script/rails server
